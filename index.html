Same Calendar Month — June, split by cohort age
The table above compares cohorts on "months since opening" (relative). This strips that out and asks: in the same calendar month, did older stores actually outperform newer ones?

All three cohorts landed within a ₹5.2K band in June, despite very different ages — the 2-3 month-old April cohort didn't meaningfully outperform stores opening for the first time that same month. If anything the May cohort (n=2) edged ahead. This suggests June's strength was more a portfolio-wide/seasonal effect than a maturity effect — worth keeping in mind when reading the ramp-up curve above as "stores get better with age."
